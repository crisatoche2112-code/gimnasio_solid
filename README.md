# Gimnasio SOLID

Aplicación web desarrollada en C# y .NET 8 para gestionar miembros, planes de
membresía, validación de accesos mediante QR o huella y registro de pagos.

## Funcionalidades actuales

- Registro y listado de miembros.
- Planes Estudiante, Regular, VIP y Fin de Semana.
- Validación de acceso mediante QR o huella.
- Cálculo de cuotas y registro de pagos.
- Historial de pagos.

## Ejecutar el proyecto

```powershell
dotnet run --project src/GimnasioSolid/GimnasioSolid.csproj
```

## Organización del trabajo

La rama `main` contiene la versión integrada del proyecto. Cada integrante debe
crear una rama propia para desarrollar su funcionalidad y realizar entre cuatro
y cinco commits antes de solicitar la integración.
